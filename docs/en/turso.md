# Turso guide moved

Origami has moved from **Turso / libSQL** to **Neon / PostgreSQL**.

Use this page instead:

- [Neon PostgreSQL detailed setup](/en/neon)

If you are upgrading an older deployment, remove these old variables:

- `TURSO_DATABASE_URL`
- `TURSO_AUTH_TOKEN`

And replace them with:

- `DATABASE_URL`
